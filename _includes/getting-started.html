Install on Mac or Linux using Homebrew:

Docker User:

Get meshplayctl

Install meshplayctl with Bash, Brew, Scoop, or download directly.

Use meshplayctl

meshplayctl system context create docker --platform docker --set meshplayctl system start {% include copy-to-clipboard.html clipboard_target="#start-helm-v3" clipboard_text="meshplayctl system context create docker --platform docker --set meshplayctl system start" %}

Use Docker Extension for Meshplay

Windows User:

Using Scoop

$ scoop bucket add meshplayctl https://github.com/khulnasoft/scoop-bucket.git $ scoop install meshplayctl $ meshplayctl system start {% include copy-to-clipboard.html clipboard_target="#start-windows" clipboard_text="scoop bucket add meshplayctl https://github.com/khulnasoft/scoop-bucket.git scoop install meshplayctl meshplayctl system start " %}

Using Binary

Not a Scoop user? Download and unzip the meshplayctl binary directly from the Meshplay releases page. Add meshplayctl to your path for ease of use. Then, execute:

$ meshplayctl system start {% include copy-to-clipboard.html clipboard_target="#start-windows" clipboard_text="meshplayctl system start" %}
