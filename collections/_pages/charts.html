Meshplay Helm Repository

Contained in the Meshplay Helm repository are two charts one of which contains many subcharts:

Chart Contents

  • Meshplay chart with subcharts for each Meshplay Adapter.
  • Meshplay Operator chart, which can be independently installed using meshplayctl, helm, or installed by Meshplay Server.

See the repository's full index of Meshplay and Meshplay Operator Helm Charts.

Helm-based Installations

  1. Install Meshplay on Kubernetes using Helm. Deploy Meshplay in Kubernetes in-cluster. (docs)
  2. Install Meshplay on Kubernetes using Meshplay CLI. Deploy Meshplay in Kubernetes in-cluster (or out-of-cluster). (docs)
  3. Connect Meshplay Server to a Kubernetes cluster and Meshplay Server will install Meshplay Operator automatically.

Customizing Meshplay’s Installation with values.yaml

Meshplay’s Helm charts support a number of configuration options. Please refer to the Meshplay Helm chart README and Meshplay Operator Helm chart README for more information.

Chart Source

The source for these charts is located in the meshplay/meshplay repository under install/kubernetes/helm.
